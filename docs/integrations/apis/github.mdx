## Authentication

GitHub supports Personal Access Tokens and OAuth. You can use either of these to authenticate with GitHub.

### Personal Access Token

To create a personal access token in GitHub, follow the instructions [here](https://docs.github.com/en/github/authenticating-to-github/creating-a-personal-access-token).

```ts my-job.ts
import { GitHub } from "@trigger.dev/github";

//create GitHub client using a token
const github = new GitHub({
  id: "github",
  token: process.env.GITHUB_TOKEN!,
});
...
```

### OAuth

To use OAuth you can connect GitHub via the Trigger.dev [web app](https://cloud.trigger.dev). Click 'Integrations' in the side panel of any project, and configure GitHub with the required ID and scopes.

```ts my-job.ts
import { GitHub } from "@trigger.dev/github";

//create GitHub client using OAuth
const github = new GitHub({
  id: "github",
});
...
```
